# EDSA Native Power Tracker

A professional, location-verified operational platform for electricity status reporting and hazard escalation in Freetown, Sierra Leone.

## 🚀 Features

- **Live Tracker**: Community-driven power status signals with automatic GPS area detection.
- **Hazard Escalation**: Report dangerous lines, falling poles, and illegal connections with photo evidence.
- **Admin Command**: Centralized dashboard for operational oversight and data export.
- **Mobile-First Design**: Premium, native-style interface optimized for field agents and citizens.

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Database**: PostgreSQL (Prisma ORM)
- **Styling**: Tailwind CSS & Framer Motion
- **Icons**: Lucide React
